# 长程任务筛选：按「diff 改动 ≥ 1000 行 且 repo ≥ 100 万行」

## 筛选标准

1. **修改行数 ≥ 1000**：patch 中 `+` 开头 与 `-` 开头的代码行数总和（去除 `+++ / ---` 文件头）
2. **仓库规模 ≥ 1,000,000 行**：`base_repo/<TASK>/repo/` 下所有文件的总行数（排除 `.git`、`node_modules`、大于 5MB 的二进制文件等）
