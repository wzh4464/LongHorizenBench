**Summary**: 本功能为 TypeScript 引入两个重要的类型系统特性：模板字面量类型（Template Literal Types）允许通过泛型占位符进行字符串字面量的拼接、转换和模式匹配；映射类型 `as` 子句（Mapped Type `as` Clauses）允许在映射类型中转换属性名称，支持过滤、重命名和生成多个属性。这些特性使得 TypeScript 能够更精确地描述字符串操作模式。

**Proposal**: 引入反引号语法的模板字面量类型表达式，支持 `${T}` 形式的占位符，实现联合类型的分布式展开和类型推断中的模式匹配；在映射类型语法中添加可选的 `as N` 子句，其中 N 是将属性名转换后的新类型，当 N 解析为 `never` 时实现过滤功能，当 N 解析为联合类型时生成多个属性。
